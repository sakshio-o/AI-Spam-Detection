# AI-Based Spam Detection

## Aim
To build an AI-based Email/SMS spam detection system using machine learning and NLP.

## Dataset
SMS Spam Collection Dataset (UCI ML Repository)

## Techniques Used
- Text preprocessing (lowercasing, tokenization)
- TF-IDF Vectorization
- Logistic Regression / Naive Bayes

## Evaluation Metrics
- Accuracy: 97.84%
- Precision: 100%
- Recall: 83.89%
- F1-score: 91.24%

## Visualizations
- Confusion Matrix
- Performance Metrics Bar Chart

## Conclusion
The model effectively detects spam messages with high accuracy and precision, making it suitable for real-world spam filtering applications.
